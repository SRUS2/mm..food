# mm..food

**mm..food** is a recipe web app built with **Flask**, **Tailwind CSS**, and the **Spoonacular API**. The name is inspired by MF DOOMâ€™s iconic album *MM..FOOD*, reflecting the blend of creativity and flavor this project brings to your kitchen.

## Features

- Search recipes by name, ingredients, or filters
- Mobile-first design inspired by Airbnb
- Bookmark your favorite recipes
- User accounts with saved recipes
- Real-time auto-suggestions via Spoonacular API

## Tech Stack

- Backend: Python, Flask
- Frontend: Tailwind CSS, HTML
- Database: PostgreSQL
- Authentication: Flask-Login
- APIs: Spoonacular API
